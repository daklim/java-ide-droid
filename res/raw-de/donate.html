JavaIDEdroid Spenden

JavaIDEdroid wird der Entwicklergemeinde kostenlos zur Verfügung gestellt.

Wenn Ihnen jedoch JavaIDEdroid gefällt und Sie Ihre Wertschätzung für all die geleistete Arbeit zeigen möchten, können Sie ein wenig Geld spenden, indem Sie JavaIDEdroidDonate kaufen.

JavaIDEdroidDonate liefert einige Debugging Informationen, welche Ihnen helfen können beim Integrieren von JavaIDEdroid in andere Android Apps. Anstelle von JavaIDEdroid starten Sie JavaIDEdroidDonate, von wo aus Sie dann JavaIDEdroid starten können. Hierbei schreibt JavaIDEdroidDonate Informationen über den aufrufenden Intent und über den von JavaIDEdroid zurückerhaltenen Antwort-Intent auf den Bildschirm. Darüber hinaus hat JavaIDEdroidDonate keine weitere eigene Funktionalität.

Sie finden JavaIDEdroidDonate im Android Market.
