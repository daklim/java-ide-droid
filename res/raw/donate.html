JavaIDEdroid Donations

JavaIDEdroid is provided free of charge to the Android developer community.

However, if you like JavaIDEdroid and want to show your appreciation for all the work that was done to create it, you can donate some money by buying JavaIDEdroidDonate.

JavaIDEdroidDonate provides some debugging information which might help you when you try to integrate JavaIDEdroid with other Android apps. Instead of starting JavaIDEdroid, you start JavaIDEdroidDonate from which you can then start JavaIDEdroid. While doing so, JavaIDEdroidDonate writes information about the calling Intent and the result Intent returned by JavaIDEdroid to the screen. Besides from that JavaIDEdroidDonate has no further functionality of its own.

You can get JavaIDEdroidDonate from the Android Market.

Many thanks for your donation!
